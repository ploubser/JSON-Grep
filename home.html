JGrep is a Ruby-based CLI tool and API for parsing and displaying JSON data using a logical expression syntax.

JGrep allows you to search a list of JSON documents and return specific documents or values based on logical truths. 


    $ cat my.json | jgrep "foo=hello and bar=world"

    {
        "foo": "hello",
        "bar": "world"
    }

JGrep also allows you to display your JSON document in a simplified manner. 


    $ cat my.json |jgrep -s foo

    hello
For more information on how Jgrep works, check out our howto page.
