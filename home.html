JGrep is a ruby based CLI tool and API for parsing and displaying json data based on a logical expression syntax.

JGrep allows you to search a list of json documents and return specific ones based on logical truths. 


    % cat my.json | jgrep "foo=hello and bar=world"

    {
        "foo": "hello",
        "bar": "world"
    }

JGrep also allows you display you json document in a simplified manner. 


    % cat my.json |jgrep -s foo

    hello
For more information on how jgrep works, check out our howto page.
