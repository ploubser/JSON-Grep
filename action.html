Real world examples

Using MCollective to check if a file is present on a host and outputs the name of the host: 


    $ mco rpc filemgr status file=/.../check_puppet.rb -j |
        jgrep 'data.md5!=a4fdf7a8cc756d0455357b37501c24b5' -s sender
    box1.example.com

Checking the total size of a log file across mutiple hosts: 


    $ mco rpc filemgr status file=/var/log/httpd/access_log -W /apache/ -j |
        jgrep -s "data.size"|
        awk '{ SUM += $1} END { print SUM/1024/1024 " MB"}'
    2757.9093 MB
